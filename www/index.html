Instructions

This application is called SmartPredict and contains a single text box with three buttons on top. Each time a user updates input in the text box, the application calls the server where a prediction algorithm will spit out three words, each with accompanying probabilities as to how likely they are to be the next word based on what the user recently types.

The most probable word will then be supplied as value of the middle button, the second and third words will then be supplied to left and right buttons respectively. You can click on these buttons to aid in fast typing, just like what you would do in conventional smart keyboard interfaces.

You can copy one of the example inputs by clicking the button beside the input text box.

About

SmartPredict is an application that utilizes HC corpora to make a model that would simulate aided word prediction via web interface. To see the code and documentation, see the codebase on this corresponding GitHub link. If you want to know more information, please send me an email here.
